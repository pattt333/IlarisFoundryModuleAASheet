# Weapon Icons Download Anleitung

## So lädst du die Icons herunter:

1. Öffne https://game-icons.net/
2. Suche nach dem Icon-Namen (z.B. "sword hilt")
3. Klicke auf das Icon
4. Klicke auf den "Download SVG" Button
5. Speichere die Datei in diesem Ordner mit dem unten angegebenen Namen
